## THE 7 FEDERAL PROTECTED CLASSES
It is illegal to discriminate based on:

1. Race
2. Color
3. National Origin
4. Religion
5. Sex including gender identity
6. Familial Status including children under 18 and pregnancy
7. Disability including physical and mental

## LANGUAGE TO NEVER USE

Category 1: Neighborhood Descriptions

"Great for families"
Use instead: "Near top-rated schools"

"Perfect for young professionals"
Use instead: "Walkable to dining and nightlife"

"Safe neighborhood"
Use instead: "[City] public data shows [X] crime index"

"Good schools"
Use instead: "[SCHOOL NAME] rated [X] out of 10 on GreatSchools.org"

"Up and coming area"
Remove entirely — implies demographic change

"Quiet established neighborhood"
Remove — can imply racial steering

"Christian community"
"Jewish community"
"Muslim community"
Remove all — never reference religion in marketing

---

Category 2: Buyer Targeting Language

"Ideal for a single person"
Reason: Familial status violation

"Perfect starter home for a young couple"
Reason: Age and familial status violation

"Adult community feel"
Reason: Familial status violation unless property is legally age-restricted

"No kids nearby"
Reason: Familial status violation

"No rentals on the block"
Reason: Source of income implication

---

Category 3: Disability Language

"Perfect for someone with mobility issues"
Reason: Disability steering — never do this

"Great for a healthy active lifestyle"
Reason: Disability implication — remove

"Handicap accessible"
Use instead: "ADA accessible" or "Wheelchair accessible"

Never describe a home as not suitable for anyone 
based on physical needs

DO include accessibility features as positives:
"Grab bars in primary bath"
"Zero-step entry"
"Roll-in shower"

---

## THE 3 QUESTION TEST
Ask these three questions about every piece of copy

Question 1:
Does this language include or exclude a protected class?
If yes — rewrite required

Question 2:
Would this copy discourage a protected class from inquiring?
Read from every protected class perspective
If any group might feel unwelcome — rewrite required

Question 3:
Are all claims verifiable and neutral?
Every neighborhood claim must cite a public data source
If not — rewrite required

## QUICK SUBSTITUTION REFERENCE

Instead of "Great school district"
Use "[School name] rated [X] out of 10 on GreatSchools.org"

Instead of "Safe neighborhood"
Use "[City] crime index: [X] per city public data"

Instead of "Great for families"
Use "Within 1 mile of 3 parks and a community pool"

Instead of "Exclusive neighborhood"
Use "Median home value of $[X] per [source]"

Instead of "Perfect for walkers"
Use "Walk Score of [X] per WalkScore.com"

Instead of "Active community"
Use "HOA includes tennis, pool, and walking trails"
